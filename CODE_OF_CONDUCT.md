# Code of Conduct

## Pledge

I want this project to be a welcoming, respectful, and productive space for everyone.
Participation in this project means agreeing to treat others with professionalism and kindness.

## Expected Behavior

- Be respectful and constructive in discussions
- Give and receive feedback in good faith
- Focus on the code and the ideas, not on people
- Be patient with contributors at different experience levels

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Discriminatory or hostile language
- Trolling or deliberately disruptive behavior

## Scope

This code of conduct applies to all project spaces.

---

This project is built to learn, collaborate, and improve together.
