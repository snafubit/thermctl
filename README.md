# TempGuard

TempGuard is a lightweight thermal monitoring tool for Linux, written in C.

The project aims to provide simple and reliable CPU temperature monitoring
with clear warning levels and minimal system overhead.

## Goals

- Provide clean CLI output
- Remain dependency-free and system-friendly
